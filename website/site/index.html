A 2d graphics library.
Built on modern ground.
(Currently in Alpha)

Lux is a 2d graphics framework written in Rust that handles window-creation , event handling , and common 2d graphics drawing operations. Simple tasks should be easy to accomplish, while more complicated functionality like custom shaders can be made possible by reaching under the hood. Lux is built on top of Glium and Glutin, both created by the excelent Tomaka.
